[js-integer-little-endian](http://aureooms.github.io/js-integer-little-endian)
==

Integer numbers code bricks for JavaScript.

```js
integer.parse( 16 , 100 , 'ff' ) ; // [ 2 , 55 ]
integer.stringify( 100 , 16 , [ 2 , 55 ] ) ; // 'ff'
```
